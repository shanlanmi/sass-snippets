## Installation

### Required
* Install [UltiSnips](https://github.com/SirVer/ultisnips)

### Installation
* Put `sass.snippet` in your chosen `snippets/` dir
* Restart Vim
* Call `:set filetype=scss` or have `filetype on` in your *.vimrc* (recommended)
